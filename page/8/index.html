Yii2 数据库迁移

发表于 2017-03-31 | 分类于 PHP ， Yii2 | | 阅读次数

描述

在开发和维护一个数据库驱动的应用程序时,数据库的结构会随代码的改变而改变. Yii 提供了一个 数据库迁移功,该功能可以记录数据库的变化, 以便使数据库和源代码一起受版本控制.
Yii 提供了一整套的迁移命令行工具，通过这些工具你可以:

  • 创建新的迁移
  • 提交迁移
  • 恢复迁移
  • 重新提交迁移
  • 现实迁移历史和状态
